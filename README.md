#### 使用说明
1.修改配置文件application.properties，数据库的连接等。

2.本地或者服务器启动时，提前开启redis服务

3.CommonController修改文件上传和下载的路径

4.同时访问后台管理系统和前台小程序时请使用两个浏览器分别访问（或者操作完一个系统退出再进行另外一个系统操作），否则会导致session冲突
